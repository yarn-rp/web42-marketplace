# homebrew-web42

Homebrew tap for the [Web42 CLI](https://web42.ai).

## Install

```bash
brew install yarn-rp/web42/web42
```

## Update

```bash
brew upgrade web42
```

## About

This tap is automatically updated by CI when a new CLI version is released.
The formula downloads pre-built binaries from [GitHub Releases](https://github.com/yarn-rp/web42-marketplace/releases).
