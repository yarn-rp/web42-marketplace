# scoop-web42

Scoop bucket for the [Web42 CLI](https://web42.ai).

## Install

```powershell
scoop bucket add web42 https://github.com/yarn-rp/scoop-web42
scoop install web42
```

## Update

```powershell
scoop update web42
```

## About

This bucket is automatically updated by CI when a new CLI version is released.
The manifest downloads pre-built binaries from [GitHub Releases](https://github.com/yarn-rp/web42-marketplace/releases).
